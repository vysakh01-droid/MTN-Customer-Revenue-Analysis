# MTN Customer & Revenue Analysis

## Project Overview
This project analyzes MTN telecom customer data to identify key business insights related to revenue, customer behavior, and churn. The analysis is performed using SQL for querying and Jupyter Notebook for data analysis and interpretation.

---

## Tools Used
- Jupyter Notebook (Python)
- SQL (PostgreSQL)

---

## Project Objectives
- Clean and prepare customer data for analysis
- Perform exploratory data analysis (EDA)
- Answer business questions using SQL queries
- Analyze revenue, churn, and customer usage patterns
- Generate insights to support business decision-making

---

## Files in This Repository
- **Python_PROJECT.ipynb** – Data analysis and SQL-based insights using Jupyter Notebook  
- **SQL_Queries.sql** – Business-focused SQL queries used in the analysis  
- **README.md** – Project overview and documentation  

---

## Key Business Questions
- What is the total and average revenue generated?
- Which subscription plans contribute the most revenue?
- What is the customer churn rate?
- How does customer tenure impact churn?
- What is the relationship between customer satisfaction and churn?
- Who are the highest revenue-generating customers?

---

## Key Insights
- Revenue distribution across subscription plans
- Identification of churned vs active customer behavior
- High-value customers contributing significantly to total revenue
- Customer tenure and satisfaction strongly linked to churn

---

## Conclusion
This project demonstrates practical data analysis skills using SQL and Jupyter Notebook. The analysis provides clear, data-driven insights that can support customer retention and revenue optimization strategies.
